# init-profile — 初回プロフィール作成

この workspace を使い始めるために、本人へ短いインタビューを行い、`context/profile.md` を初期化する。

## 手順

1. `AGENTS.md` と `context/INDEX.md` を読む。
2. まだ `context/profile.md` が未初期化であることを確認する。
3. ユーザーへ、必要最小限の質問をまとめて聞く。質問は多くても10個程度にする。
   - 呼び方
   - 主な役割・活動
   - AIに任せたい仕事
   - AIに任せたくない仕事
   - よく扱う資料・成果物
   - 文章の好み
   - この repo に入れてよい情報 / 入れてはいけない情報
   - commit / push を自動で行ってよいか
4. 回答をもとに `context/profile.md` を更新する。
5. `context/INDEX.md` のプロフィール行を実態に合わせる。
6. 作業記録を `minutes/YYYYMMDD_初期プロフィール作成/README.md` に残す。

## 制約

- 秘密情報や認証情報は聞かない。
- 第三者の機微情報は書かない。
- 不明な項目は空欄か「未確認」として残す。
